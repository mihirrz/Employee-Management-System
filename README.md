# Employee Management System

A simple Employee Management System built using **Spring Boot** and **Java**. This system allows you to add, retrieve, and delete employee records through a RESTful API. The project is designed to help HR departments or teams manage employee details such as name, level, and status.

## Features

- **Add an Employee**: Add new employee records with basic details such as name, job level, and status.
- **Get Employee by ID**: Retrieve the details of a specific employee by their ID.
- **Get All Employees**: Fetch all employee records stored in the database.
- **Delete Employee**: Remove an employee record from the system by ID.

## Technologies Used

- **Spring Boot** – A framework used for building the back-end services.
- **Spring Data JPA** – To interact with the database.
- **H2 Database** (default) – In-memory database to store employee data. You can switch to other databases like MySQL or PostgreSQL.
- **Postman** – For testing API endpoints.
- **Java** – The programming language used to build the application.
- **Maven** – Build tool for managing project dependencies.
